# The West Super Secret Scripts

## Instalace

Vytvořit v prohlížeči novou oblíbenou stránku (klasický bookmark) a jako adresu vložit:

```sh
javascript:(function(){document.body.appendChild(document.createElement('script')).src='https://deprivant.github.io/tw-super-secret-scripts/dist/tw-super-secret-scripts.min.js';})();
```
